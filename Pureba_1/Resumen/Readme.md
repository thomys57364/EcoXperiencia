# 📁 Estructura del proyecto creada
```
ecoxperiencia/
  ├── index.html              → Inicio con Hero, categorías, experiencias destacadas
  ├── explorar.html           → Listado con filtros por categoría, precio, ubicación
  ├── experiencia.html        → Detalle completo con galería, reservas, reseñas
  ├── como-funciona.html      → Explicación del proceso y beneficios
  ├── anfitrion.html          → Página para convertirse en anfitrión
  ├── login.html              → Inicio de sesión
  ├── registro.html           → Registro de usuarios
  ├── perfil.html             → Perfil de usuario editable
  ├── favoritos.html          → Experiencias guardadas
  ├── contacto.html           → Formulario de contacto funcional
  ├── legal.html              → Términos y privacidad
  ├── soporte.html            → FAQ y centro de ayuda
  ├── 404.html                → Página no encontrada
  ├── css/
  │   └── styles.css          → Todos los estilos del sitio
  └── js/
      └── app.js              → Lógica completa de la aplicación
```
# ✅ Funcionalidades implementadas

| Funcionalidad | Descripción |
|---------------|-------------|
| Navegación | Todas las páginas interconectadas con navbar y footer reutilizables |
| Auth localStorage | Registro, login, logout y gestión de sesión |
| Favoritos | Guardar/eliminar con localStorage, requiere login |
| Buscador Hero | Redirige a explorar.html con parámetros en URL |
| Filtros | Por categoría, precio, ubicación y texto |
| Detalle experiencia | Galería, reservas con selección de fechas/personas |
| Perfil usuario | Editable, muestra avatar, logout |
| Contacto | Formulario con simulación de envío |
| Toast notifications | Mensajes de éxito/error |
| Modales | Confirmaciones y alertas |
| Mobile menu | Menú hamburguesa funcional |

# 🎨 Diseño mantenido
 - Paleta eco-luxury original (verde musgo, dorado, crema)
 - Tipografías Playfair Display y DM Sans
 - Animaciones fade-up y transiciones suaves
 - Responsive design completo
 - Sombras, bordes y espaciado consistente
